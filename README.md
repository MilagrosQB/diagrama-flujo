Diagrama de flujo
===================


Pedir comida por delivery
----------
1. Inicio
2. Decidir que comida comprar
3. Llamar al restaurante de preferencia 
4. Consultar precios u ofertas
5. ¿Aceptar las ofertas?
6. Brindar datos 
7. ¿Confirmación total?
8. Recepción de comida
9. Pago
10. Fin
